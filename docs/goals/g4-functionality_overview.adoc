[[G4]]
=== G.4 Functionality Overview

NOTE: Overview of the functions (behavior) of the system. Principal properties only
(details are specified in the System book) (see {Handbook}).

The Smart Home System integrates a set of high-level functionalities designed to
ensure security, occupant safety, assistance in daily activities, and efficient
energy management. These functionalities enable the benefits described in G.3 and
address the issues identified in G.2.

The system combines physical security, cybersecurity, adaptive assistance, and
energy optimization within a unified intelligent environment.


//---- High-Level Functionality
[[g4-security-prevention]]
^`G.4.{counter:g4}`^  
Prevent unauthorized access through integrated physical and digital security
mechanisms.
//----

//---- High-Level Functionality
[[g4-access-control]]
^`G.4.{counter:g4}`^  
Control access to the dwelling using identification and authorization mechanisms.
//----

//---- High-Level Functionality
[[g4-hazard-detection]]
^`G.4.{counter:g4}`^  
Detect dangerous situations such as intrusions, fires, or accidental falls.
//----

//---- High-Level Functionality
[[g4-emergency-alerting]]
^`G.4.{counter:g4}`^  
Automatically alert emergency services when critical situations are detected.
//----

//---- High-Level Functionality
[[g4-data-protection]]
^`G.4.{counter:g4}`^  
Protect personal and system data by ensuring secure access to system services.
//----

//---- High-Level Functionality
[[g4-adaptive-assistance]]
^`G.4.{counter:g4}`^  
Provide adaptive assistance and support for occupants based on their profiles,

//----

//---- High-Level Functionality
